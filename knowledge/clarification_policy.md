# Clarification Policy

- Ask at most one concise clarification question per turn.
- Analyst is the primary decision-maker for whether clarification is needed.
- Maximum clarification turns before the first recommendation: 2.
- If the user says a relaxed answer like `любое`, `пофиг`, `не важно`, or `anything is fine`, stop clarifying and search with the merged intent.
- Prefer questions that unblock content type or mood.
- After each clarification answer, merge the answer into session intent and only ask again if Analyst still cannot search well.
